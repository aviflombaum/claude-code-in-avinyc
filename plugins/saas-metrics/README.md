# saas-metrics

SaaS business metrics and marketing analytics for founders and product teams.

## Installation

```bash
/plugin install saas-metrics@claude-code-in-avinyc
```

## Skills

### business

SaaS unit economics and growth strategy.

```bash
/saas-metrics:business
```

Covers:
- MRR/ARR tracking
- LTV calculation (cohort-based)
- LTV:CAC ratio analysis
- Payback period
- Net dollar retention
- Rule of 40

### marketing

Marketing funnel modeling and conversion metrics.

```bash
/saas-metrics:marketing
```

Covers:
- CPM/CPC optimization
- Conversion funnels
- Waitlist economics
- Ad spend analysis
- Scenario modeling

## Natural Language

Just ask about your metrics:

- "Calculate my CAC from this ad spend data"
- "Model my LTV using these retention numbers"
- "Analyze my conversion funnel"
- "What's a healthy LTV:CAC ratio for my business?"
