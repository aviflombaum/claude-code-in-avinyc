# design-system

UI/UX design expertise combining usability principles with visual implementation.

## Installation

```bash
/plugin install design-system@claude-code-in-avinyc
```

## Skills

### ux-ui

UX/UI design principles for clean, intuitive interfaces.

```bash
/design-system:ux-ui
```

Covers:
- Information architecture
- Interaction design
- Accessibility (WCAG)
- Responsive design
- Performance and efficiency

### web-designer

Web graphic design for interfaces, layouts, and visual systems.

```bash
/design-system:web-designer landing page
/design-system:web-designer dashboard
/design-system:web-designer bauhaus style
```

Covers:
- Visual hierarchy
- Layout and grid systems
- Typography
- Color and contrast
- Style translation (Bauhaus, Pop Art, Retro, Futuristic)

## Natural Language

Just describe your design needs:

- "Design a clean dashboard layout"
- "Improve the usability of this form"
- "Create a landing page with a futuristic aesthetic"
- "Review this interface for accessibility"

## When to Use Each Skill

| Need | Skill |
|------|-------|
| Usability review | ux-ui |
| Information architecture | ux-ui |
| Accessibility audit | ux-ui |
| Visual design direction | web-designer |
| Landing page design | web-designer |
| Style/aesthetic guidance | web-designer |
