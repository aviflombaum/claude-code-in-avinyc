# secure-server

> **Status: In Development** - Not yet published to the marketplace.

Ubuntu server hardening plugin for Claude Code. Configures fail2ban, UFW, auditd, rkhunter, unattended-upgrades, and custom alerting.

## Skills

- `/secure-server [server hostname or IP]` - Harden Ubuntu servers with security tools and alerting

## Commands

- `/secure-server` - Invoke the secure-server skill
