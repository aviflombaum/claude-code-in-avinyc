# Claude Code in AviNYC

A Claude Code plugin marketplace for Ruby and Rails development.

## Installation

Add this marketplace to Claude Code:

```bash
/plugin marketplace add aviflombaum/claude-code-in-avinyc
```

## Available Plugins

### rspec-writer

AI-powered RSpec test generation for Rails applications.

**Install:**
```bash
/plugin install rspec-writer@claude-code-in-avinyc
```

**Usage:**
```bash
# Slash command
/rspec-writer:write-test model User

# Or just ask naturally
"Write tests for the User model"
```

**Supports:**
- Model specs
- Request specs
- System specs
- Job specs
- Mailer specs
- Channel specs
- ActiveStorage specs
